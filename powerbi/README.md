# Power BI dashboard plan

1. Import the cleaned CSV from the data folder into Power BI.
2. Use the following visuals:
   - KPI cards for Total Sales, Total Profit, Total Orders, and Average Order Value.
   - Line chart for Monthly Sales Trend.
   - Bar chart for Top Products.
   - Stacked bar chart for Category-wise Sales.
   - Map or clustered bar chart for Regional Sales.
   - Matrix/table for Profit Analysis.
   - Slicer for Region, Category, and Customer.
3. Save the report as retail_business_insights.pbix in this folder.
